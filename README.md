# 기상청 전자민원 해상 기상특보 자동 수집

기상청 전자민원(minwon.kma.go.kr)에서 해상 기상특보(태풍주의보·풍랑주의보) 과거
자료를 연도별로 신청하고 PDF까지 자동 저장하는 Playwright 스크립트입니다.

한 번에 신청 가능한 용량 제한 때문에 구역을 나눠 반복 신청해야 하는데, 그 반복을
자동화했습니다. 신청 직후 발급 PDF를 규칙에 맞는 폴더/파일명으로 내려받습니다.

---

## 파일은 딱 두 개

| 파일 | 언제 쓰나 | 바꾸는 것 |
|---|---|---|
| `kma_minwon_auto.py` | **개편이 안 걸친 모든 연도** (대부분) | `YEARS`, `ZONE_GROUPS` |
| `kma_minwon_auto_split.py` | **개편이 걸친 연도의 먼바다 보완** (2021·2019·2014만) | `YEARS`, `PERIOD_END`, `FILE_TAG`, `ZONE_GROUPS` |

두 파일의 실행 로직은 완전히 동일합니다. 상단 설정 값만 다릅니다.
아래 블록을 복사해 해당 파일 상단에 붙여넣으면 원하는 연도로 동작합니다.

> **왜 파일이 두 개인가?** 해상 "먼바다" 구역명이 과거 세 번(2014.12.10 / 2019.04.30 /
> 2021.07.29) 개편됐습니다. 개편일이 한 해에 걸치면 그 해 먼바다 자료가 개편일을
> 경계로 두 조각으로 나뉘어, 옛명/현행명을 기간을 쪼개 각각 신청해야 온전한 1년이
> 됩니다. 그 "개편 전 조각"을 받는 게 `_split` 파일입니다. 앞바다는 개편 무관이라
> 항상 기본 파일로 1년 통짜입니다.

---

## 처음 한 번 세팅

```
conda create -n kma python=3.11 -y
conda activate kma
pip install playwright
```

작업 폴더에 두 스크립트를 두고, 스크립트 상단 `DOWNLOAD_ROOT`(PDF 저장 위치)와
`BOARD_URL`(민원보관함 주소)을 환경에 맞게 확인하세요.

## 매번 실행 절차

1. 엣지를 전부 닫고(작업관리자에서 `msedge.exe` 잔여 프로세스까지 종료), 새 cmd에서:
   ```
   "C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe" --remote-debugging-port=9222 --user-data-dir="C:\edge_automation"
   ```
   경로 오류 시 ` (x86)`를 뺀 경로로 시도.
2. 뜬 엣지 창에서 전자민원 접속 → **PASS 로그인**. 이 창은 작업 끝까지 건드리지 않음.
3. 다른 프롬프트에서:
   ```
   conda activate kma
   python kma_minwon_auto.py        (또는 kma_minwon_auto_split.py)
   ```

## MODE (두 파일 공통)

- `dump` : 사이트의 구역 체크박스 이름 전체 출력 (ZONE_GROUPS 검수용)
- `dry`  : 확인 페이지까지 리허설, 신청/다운로드 안 함 (실전 전 필수 확인)
- `run`  : 실제 신청 + PDF 자동 다운로드

새 연도를 처음 돌릴 땐 **dry로 검증 통과 → 1건만 run 테스트 → 전체 run** 순서를 권장합니다.

# 연도별 복붙 블록

## A. 개편 안 걸친 연도 → `kma_minwon_auto.py`

`YEARS`만 바꾸면 되는 경우와, `ZONE_GROUPS`까지 바꿔야 하는 경우가 있습니다.
먼바다 구역명 체계가 시대별로 다르기 때문입니다.

### A-1. 2022 ~ 현재 (현행 구역명)

`ZONE_GROUPS`는 파일에 기본 내장된 현행 30구역을 그대로 사용. `YEARS`만 교체.

```python
YEARS = [2024]   # 원하는 해로. 2022 이상 현재까지
# ZONE_GROUPS 는 파일 기본값(현행명) 그대로 둔다
```

### A-2. 2020 (옛 ~'21.07.29 체계, 1년)

YEARS = [2020]

ZONE_GROUPS = {

### A-3. 2015 ~ 2018 (동해남부·서해남부 먼바다만 ~'19.04.30 통짜)

A-2에서 **동해남부·서해남부 먼바다 두 줄만** 아래로 교체(나머지는 A-2와 동일):

```python
    ("동해남부", "먼바다"): ["동해남부먼바다(~'19.04.30)"],
    ("서해남부", "먼바다"): ["서해남부먼바다(~'19.04.30)"],
```

### A-4. 2013 이하 (남해서부 먼바다까지 ~'14.12.10 통짜)

A-3에서 추가로 **남해서부 먼바다 줄을** 아래로 교체:

```python
    ("남해서부", "먼바다"): ["남해서부먼바다(~'14.12.10)"],
```

> A-2/A-3/A-4는 앞바다 구역은 모두 동일합니다. 먼바다 구역명만 시대에 맞게 달라집니다.
> 새 연도를 처음 돌리기 전 반드시 `MODE="dump"`로 실제 사이트 구역명과 대조하세요.

## B. 개편 걸친 연도 → `kma_minwon_auto_split.py`

이 파일은 해당 연도의 **먼바다 "개편 전 조각"만** 신청합니다. 그 연도의 앞바다와,
개편과 무관한 다른 해역 먼바다는 A(기본 파일)로 따로 수집해야 합니다.

각 블록의 `YEARS`, `PERIOD_END`, `FILE_TAG`, `ZONE_GROUPS`를 통째로 붙여넣으세요.

### B-1. 2021 (개편일 7.29, 남해서부 제외 6개 해역)

```python
YEARS = [2021]
PERIOD_END = "0729"
FILE_TAG   = "_0101-0729"

ZONE_GROUPS = {
    ("동해중부", "먼바다"): ["동해중부먼바다(~'21.07.29)"],
    ("동해남부", "먼바다"): ["동해남부북쪽먼바다(~'21.07.29)", "동해남부남쪽먼바다(~'21.07.29)"],
    ("서해중부", "먼바다"): ["서해중부먼바다(~'21.07.29)"],
    ("서해남부", "먼바다"): ["서해남부북쪽먼바다(~'21.07.29)", "서해남부남쪽먼바다(~'21.07.29)"],
    ("남해동부", "먼바다"): ["남해동부먼바다(~'21.07.29)"],
    ("제주도", "먼바다"): ["제주도남쪽먼바다(~'21.07.29)"],
}
```

### B-2. 2019 (개편일 4.30, 동해남부·서해남부만)

```python
YEARS = [2019]
PERIOD_END = "0430"
FILE_TAG   = "_0101-0430"

ZONE_GROUPS = {
    ("동해남부", "먼바다"): ["동해남부먼바다(~'19.04.30)"],
    ("서해남부", "먼바다"): ["서해남부먼바다(~'19.04.30)"],
}
```

### B-3. 2014 (개편일 12.10, 남해서부만)

```python
YEARS = [2014]
PERIOD_END = "1210"
FILE_TAG   = "_0101-1210"

ZONE_GROUPS = {
    ("남해서부", "먼바다"): ["남해서부먼바다(~'14.12.10)"],
}
```

> **2014 특이점**: 개편일이 연말이라 "개편 전 조각"이 1.1~12.10로 거의 1년, "개편 후
> 조각"이 12.11~12.31로 약 3주입니다. 분량이 비대칭이지만 정상입니다.

---

## 개편 걸친 연도의 나머지 절반(하반기 조각) 처리

`_split`으로 받은 건 "개편 전" 조각입니다. "개편 후" 조각은 기본 파일(A)이 그 연도를
1년 통짜로 돌릴 때 함께 받아지는데, 이때 파일명에 꼬리표가 없으므로 상·하반기 짝을
맞추려면 하반기 쪽에 꼬리표를 따로 붙여야 합니다.

- 2021 하반기: `_0730-1231`
- 2019 하반기: `_0501-1231` (동해남부·서해남부 먼바다에만)
- 2014 하반기: `_1211-1231` (남해서부 먼바다에만)

2019·2014는 같은 연도의 다른 파일엔 꼬리표가 없어야 정상이라, 일괄 rename이 아니라
**해당 해역 먼바다 파일만 수동으로** 붙이는 걸 권장합니다.

## 저장 규칙

```
폴더:   {DOWNLOAD_ROOT}\{연도}\{해역}\{앞바다|먼바다}\
파일명: 기상특보(해상)_{연도}_{해역}_{구분}_{특보}[_기간꼬리표].pdf
```

- 폴더명은 `(1)(2)` 꼬리를 뗀 통일형, 파일명은 `(1)(2)` 유지
  (서해남부 앞바다(1)(2)는 같은 `앞바다` 폴더에 공존, 파일명으로 구분)
- 기간 꼬리표는 개편 걸친 연도의 먼바다에만 붙음. 그 외엔 꼬리표 없는 1년 통짜.

---

## 주의사항

- **개편 걸친 연도(2021·2019·2014)를 기본 파일로 돌리면 안 됨** — 먼바다가 반쪽만
  나옵니다. 반드시 기본(앞바다+무관 해역) + `_split`(개편 해역 먼바다) 조합으로.
- 발급은 신청 후 7일까지만 가능 → 신청한 연도는 바로 다운로드까지 완료(스크립트가
  신청 직후 자동 다운로드하므로 보통 문제 없음).
- 정부 사이트 세션은 약 30분 후 만료될 수 있음 → 연도 단위로 나눠 실행 권장.
- 신청 사이 간격(`DELAY_SEC`, 기본 4초)은 서버 부하 방지용 → 줄이지 말 것.
- 신청 이력은 `신청로그.csv`에 자동 기록됩니다.
